# Bootcamp HTML Inicial | SomosF5

Este repositorio contiene mi primer ejercicio práctico de **HTML5** realizado durante el bootcamp **AI4Inclusion** de la Fundación Somos F5 (edición 2026).

## 🎯 Sobre el proyecto
El objetivo principal fue comprender cómo organizar el contenido de manera jerárquica y eficiente, aplicando las mejores prácticas recomendadas 

## 🔍 Características del Código
*   **Semántica:** Uso correcto de etiquetas HTML5 (`<header>`, `<main>`, `<section>`, `<footer>`) para mejorar la accesibilidad y el SEO.
*   **Documentación interna:** El código incluye comentarios explicativos detallados en cada bloque, sirviendo como guía de aprendizaje sobre el propósito de cada etiqueta.
*   **Enfoque Educativo:** Diseñado bajo una lógica estructurada, reflejando mi enfoque científico aplicado al código.
